Netflix Movie Data Analysis
This project performs an in-depth Exploratory Data Analysis (EDA) on a dataset of movies (sourced from mymoviedb.csv) to uncover trends in genres, popularity, and release years. The analysis aims to answer specific business and entertainment questions using Python's data science ecosystem.

🚀 Project Objectives
The analysis focuses on answering the following key questions:

Genre Frequency: Which genres appear most frequently in the dataset?

Popularity vs. Genre: Which genres garner the highest votes and fan engagement?

Top Performers: Identifying the most and least popular movies and their associated genres.

Release Trends: Determining which year had the highest rate of movie releases.

📊 Key Findings
Dominant Genre: The Drama genre is the most frequent, appearing in over 14% of the dataset.

Fan Favorites: Drama also leads in popularity, accounting for 18.5% of the most highly-voted movies.

Top Movie: Spider-Man: No Way Home holds the highest popularity rating, classified under Action, Adventure, and Science Fiction.

Peak Production Year: The year 2020 saw the highest filming rate within this dataset.

🛠️ Requirements & Dependencies
To run this analysis, you need Python and the following libraries:

pandas - Data manipulation

numpy - Numerical operations

matplotlib - Basic plotting

seaborn - Advanced statistical visualization

Install them via pip:

Bash
pip install pandas numpy matplotlib seaborn
📖 How to Run
Dataset: Ensure mymoviedb.csv is in the same directory as the notebook.

Execute: Run the cells in Netflix Data Analysis.ipynb to generate the summary statistics and visualizations.

Cleaning: The notebook includes steps to handle lineterminator issues and data formatting to ensure accurate results.

📈 Visualizations
The project utilizes Seaborn and Matplotlib to create:

Distribution plots for movie popularity.

Frequency charts for genres.

Time-series analysis for movie release years.
